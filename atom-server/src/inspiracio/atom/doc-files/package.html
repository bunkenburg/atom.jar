This package implements Atom service. These classes speak about business logic. The implementation of Atom is in package atom.

This package requires Java 5. It calls our other classes, which still only require Java 1.4.2.

The classes in this package receive Atom http requests, parse and interpret the GData URL parameters, delegate the calls to some LAOs, and transform the results into XML. The central class is AtomServlet. The most important classes are:

  • AtomServlet. Serves Atom requests.
  • Parameters. The URL parameters for an Atom request.

The Package inspiracio.atom.offer implements the Atom API for job offer.
